Batch Sequencing Visualization

This application visualizes the ETL (Extract, Transform, Load) batch sequencing for the Health Catalyst DOS Platform. It helps you understand the dependencies and execution flow from Source Mart jobs to SAM Jobs.



Features:

  • All Batches: View the complete batch sequence diagram
  • Did Not Run Today: Filter to show only batches that have not yet run today or failed.
  • Broken References: View batches with broken references/dependencies.
    Note: Did Not Run Today / Broken References, include parent dependencies for lineage purposes.
  • History: Access historical snapshots to compare batch sequences over time
  • Zoom & Pan: Use mouse wheel to zoom, click and drag on whitespace to pan
  • Text Selection: Click on node text to select and copy information
